# _Zestra_
_Where Ragas and Rhythms Converge. Embark on a Harmonious Sonic Adventure!_


### _Run Locally_

_Clone the project_

```bash
  git clone https://github.com/heyyrxjesh/Zestra.git
```

_Go to the project directory_

```bash
  cd Zestra
```

_Install dependencies_

```bash
  npm install
```

_Start the server_

```bash
  npm run dev
```
